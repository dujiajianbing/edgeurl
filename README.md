# edgeurl
make a edge extension
## manifest.json
扩展的清单文件，定义扩展的属性和权限。
## popup.html
弹出窗口页面，包含按钮来触发复制和转换操作。
## popup.js
处理按钮点击事件的脚本
## 使用方法
- 将edgeurl文件夹下载到本地
- 打开 Edge 浏览器，输入 edge://extensions 进入扩展管理页面。
- 启用 "开发者模式"，点击页面右上角的开关按钮。
- 点击 "加载已解压的扩展"，选择之前下载的文件夹，然后点击 "选择文件夹"。
- 扩展将会被加载到 Edge 浏览器中，并在扩展栏中显示。
